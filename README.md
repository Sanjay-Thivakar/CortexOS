# CortexOS

CortexOS is an AI-powered cognitive operating system designed for students and developers.

It unifies notes, code repositories, and tasks into a single intelligent platform with:

- Semantic search across knowledge
- Smart daily focus generation
- Context-aware task creation
- Automatic code-notes linking
- GitHub integration
